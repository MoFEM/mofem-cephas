Password and login

  • On the first login, select the password
  • The shell password on the SSH login is different than the JupyterHub password
  • Ask admin for SSH password
  • Change password in the shell using passwd command
  • Set up your shell with the command (e.g. bash):
    $ usermod --shell /bin/bash your_login_name

SSH config

Copy the following code into .ssh/config on your laptop or desktop.

Host greenstick
    HostName greenstick.eng.gla.ac.uk
    ForwardX11 yes
    Compression yes
    User YOUR_LOGIB
    LocalForward 2222 greenstick.eng.gla.ac.uk:2222
    LocalForward 9000 greenstick.eng.gla.ac.uk:9000
  
  Host forward_greenstick
    HostName localhost
    ForwardX11 yes
    Compression yes
    User YOUR LOGIN
    Port 2222

Fisrt you need to make ssh connection with server. That is:

ssh greenstick

which forward ports of greenstick to your local machine. Then you can use address http://localhost:9000 to connect to hab, and in VS code you choose forward_greenstick.
